可变参数

有些方法的语义要求它们必须能接受个数可变的实参——例如著名的main方法，就需要能接受所有的命令行参数为实参，而命令行参数的数目，事先根本无法确定下来。对于这个问题，传统上一般是采用“利用一个数组来包裹要传递的实参”的做法来应付。大体说来，“Varargs”是“variable number of arguments”的意思。

  • 若除了可变长参数还有其它参数，可变长参数一定要放在最后
